## Installation

Clone this repo 
```
git clone https://github.com/aquatome/ImageRecognition.git
```

Make the build.sh file executable
```
chmod +x build.sh
```

Launch the executable build.sh
```
./build.sh
```

To clean the folder
```
make clean
```
